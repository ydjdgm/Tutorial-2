# 계획
캐주얼 액션 게임

공을 굴려서 탬 먹고 골인 지점까지 도달하기

구성:

공(Player), 템(Item), 지형(Platform), 장애물(Obstacle), 결승점(Point)

공을 굴려서
제한시간 안에
아이템을 먹고
결승점에 도달하면
클리어

제한시간이 지나면 실패

아이템을 다 먹어야만 결승점이 활성화 됨

아이템 먹기, 클리어 실패시 리젠

게임 클리어 로직

아이템 다 먹고 Point 도달 시 게임 클리어

UI제작:

먹은아이템/최대아이템 표시

제한시간 관련 로직; 제한시간 <= 0이면 gameStatus = 0으로

UI 제한시간 표시
